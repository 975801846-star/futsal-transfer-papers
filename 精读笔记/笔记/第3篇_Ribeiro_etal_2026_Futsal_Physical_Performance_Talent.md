# 精读 #3: Impact of Physical Match Performance on Youth Futsal Talent Program Selection: A Detailed Analysis by Team Rank in Portugal

*Research Quarterly for Exercise and Sport*, 97(1): 5–12

## 1. 研究问题与设计

**研究目的：** 检验葡萄牙 U-15 男子五人制球员的比赛体能表现（LPS 追踪）与①球队全国排名（R1/R2/R3）和②国家队发展计划（NTDP）选材结果之间的关系。

**设计类型：** 横截面观察研究，嵌套于为期 4 天的 U-15 地区协会间锦标赛。

| 参数 | 数值 |
|------|------|
| 样本 | n=202 名男子 U-15 球员，来自 22 支球队 |
| 追踪设备 | Local Positioning System (LPS) |
| 球队分级 | R1 (top) / R2 (middle) / R3 (bottom) 三档全国排名 |
| 速度分区 | LIR: 0–12 km/h, MIR: 12.1–18 km/h, HIR: 18.1–30 km/h |
| 因变量 | TD (总距离), 各速度区间的相对强度 (m/min) 和占比 (%) |
| 统计方法 | Linear Mixed Models |

---

## 2. 核心发现

### 2.1 球队排名效应

| 比较 | 关键指标 | p 值 | Effect Size |
|------|----------|------|-------------|
| R1/R2 vs R3 | MIR 占比 | ≤.001 | 0.4–1.1 |
| R1/R2 vs R3 | HIR 占比 | ≤.001 | 0.4–1.1 |
| R1/R2 vs R3 | 相对强度 (m/min) | ≤.001 | — |

→ 高排名球队（R1/R2）在 **中高强度跑占比** 和 **总体相对强度** 上显著优于 R3，效应量达中等～大。

### 2.2 NTDP 选材效应（⭐ 核心）

| 比较 | 关键指标 | p 值 | Effect Size |
|------|----------|------|-------------|
| 入选 vs 未入选 | 绝对总距离 (TD) | ≤.01 | 0.7 |
| 入选 vs 未入选 | LIR 相对强度 | ≤.01 | — |
| 入选 vs 未入选 | MIR 相对强度 | ≤.01 | — |
| 入选 vs 未入选 | **HIR 相对强度** | ≤.01 | **0.7–1.7** |

→ **HIR（高强度跑）是区分 NTDP 入选与未入选的核心体能指标，效应量最大达 1.7**。

## 3. 方法论评析

### 3.1 优势

1. **生态效度高：** 使用真实锦标赛环境（4 天地区间比赛）而非测试场景，LPS 数据直接反映比赛需求。
2. **速度分区合理：** HIR 阈值 (18.1 km/h) 切合五人制比赛的高频变速特征；若按足球 19.8 km/h 标准会丢失敏感信息。
3. **双因变量设计：** 同时考察排名（团队层面）和 NTDP 入选（个体层面），提供了从宏观到微观的体能−选材证据链。
4. **Linear Mixed Models：** 考虑了球员嵌套于球队的层级结构，避免了假阳性膨胀。
5. **葡萄牙五人制生态：** 葡萄牙是全球五人制强国（FIFA 排名第 5），其 NTDP 选材体系具外部参考价值。

### 3.2 局限

1. **横截面设计：** 单次锦标赛无法建立因果方向——是体能好→入选，还是入选激励→体能表现更好？需纵向追踪。
2. **仅体能维度：** 没有同时采集技术（传球成功率、控球）和战术（决策）指标，无法评估体能对选材的独立贡献 vs 多维交互。
3. **年龄局限：** U-15 正值青春期，体格发育差异（生物成熟度）可能混淆体能−选材关系。未报告 maturity offset 是明显遗漏。
4. **选材标准未知：** 论文未披露 NTDP 选拔的具体 criteria（体能、技术、战术权重），使 HIR-TD 的预测效度解读受限。
5. **效应量解释需谨慎：** ES=1.7 虽属 large，但若置信区间宽（未报告），在小样本子组中可能不稳定。

## 4. 对本综述的贡献定位

### 4.1 与"五人制→足球技能迁移"框架的关系

| 框架维度 | 本研究的贡献 |
|----------|-------------|
| **人才识别（Talent ID）** | ⭐⭐⭐ 直接贡献：首次量化五人制比赛中 HIR 对国家队选材的预测效应 |
| **供体运动框架（Donor Sport）** | ⭐⭐ 间接近：为五人制作为足球"供体运动"提供了体能维度的区分效度证据 |
| **决策/技能迁移** | ⭐ 弱相关：未涉及技术-战术指标，仅体能层面 |

### 4.2 关键启示

> 如果五人制 U-15 的 HIR 表现能区分 NTDP 入选，那这些高强度跑能力可被视为 **体能先决条件（physical prerequisite）**，为后续技战术迁移提供生理基础。换言之，五人制练出的高变速跑动能力 = 进入足球人才管道的"入场券"之一。

### 4.3 可引用的关键句

- _"Players selected for the NTDP demonstrated superior absolute total distance (TD) and relative intensities across various running categories (LIR, MIR, and HIR) compared to non-selected players (p ≤ .01; ES: 0.7–1.7)."_
- _"These findings underscore the correlation between higher performance metrics and team rankings, emphasizing the need for targeted training to enhance HIR performance."_
