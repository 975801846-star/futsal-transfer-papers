# 🚣 学舟出题 — 使用说明

## 这个文件夹是干什么的？

你需要喂给学舟生成题目的所有素材都在这里。**其他文件夹（论文库/、精读笔记/、研究方案/）是论文写作参考，不需要在学舟里操作。**

## 三个子文件夹

| 文件夹 | 用途 | 怎么用 |
|--------|------|--------|
| **复习资料/** | 结构化知识点 | 整篇复制 → 学舟「文本输入」→ 生成综合题 |
| **每周题目/** | Every Saturday cron 自动产出 | 查看本周题目即可，也会同步产出 StudyOS JSON |
| **论文精要/** | 单篇论文的结构化摘要 | 选一篇 → 复制全文 → 学舟 `/api/quizzes/generate` → 生成针对该论文的题目 |

## 学舟出题参数建议

根据论文精要的内容深度，推荐使用以下参数：

| 精要难度标注 | question_types | count | difficulty |
|-------------|----------------|-------|------------|
| basic | [choice, tf] | 5 | basic |
| advanced | [choice, tf, short_answer] | 5-8 | advanced |
| comprehensive | [choice, tf, short_answer, case_study] | 8-10 | comprehensive |

## 更新方式

- **每周题目**：每六上午 10:00 自动生成（cron: `4f5a3271`）
- **论文精要**：有新论文精读完成后，手动添加精要到此文件夹
- **复习资料**：知识体系更新后手动更新
