```
(futsal OR "indoor soccer" OR "five-a-side") AND ("skill transfer" OR "donor sport" OR "talent development") AND (football OR soccer)
```

## curl 命令行速查

### PubMed
```bash
curl -s "https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi?db=pubmed&retmax=10&term=futsal+skill+transfer+football" | grep "<Id>" | sed 's/.*<Id>\(.*\)<\/Id>/\1/'
```

### Semantic Scholar
```bash
curl -s "https://api.semanticscholar.org/graph/v1/paper/search?query=futsal+skill+transfer+football&limit=10&fields=title,year,authors,publicationVenue,openAccessPdf" 
```

### OpenAlex
```bash
curl -s "https://api.openalex.org/works?search=futsal+skill+transfer+football&per_page=10&sort=publication_date:desc"
```

### CORE.ac.uk
```bash
curl -s "https://api.core.ac.uk/v3/search/works?q=futsal+football+skill+transfer&limit=10"
```

## 周一 cron 多源搜索流程（建议）

```
1. PubMed E-utilities → 搜索核心 + P0 纵向 + P0 负迁移 → 各取 top 5
2. Semantic Scholar API → 同上 → 取 top 5（带 API key 防限流）
3. 合并去重 → 按年份排序 → 取最新的 5 篇
4. OpenAlex → 补充确认 5 篇的来源和 OA 状态
5. CORE.ac.uk → 尝试获取 5 篇的 PDF URL
6. 有 PDF → 下载到全文PDF/；无 PDF → 记录「付费墙」
7. 精读第 1 篇 → 写入摘要精读/
8. 更新 README 索引
9. git commit + push
```
