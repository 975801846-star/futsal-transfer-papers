## 一、研究问题与假设

### 核心问题
**五人制足球经验对十一人制足球表现的迁移效应是什么？迁移效应的异质性如何解释？**

### 子问题（4 个可独立发表的研究）
| # | 研究问题 | 对应方法学 | 预期论文类型 |
|---|----------|-----------|-------------|
| RQ1 | 有无五人制背景的球员在十一人制技战术指标上有何差异？ | 组间对比 + 情境变量控制 | 实证研究 |
| RQ2 | 五人制经验的哪些维度（年限/水平/位置/转项年龄）对迁移效应贡献最大？ | 可解释 ML（SHAP） | 方法学+实证 |
| RQ3 | 迁移效应是否存在不同的子类型（技术迁移型 vs 决策迁移型 vs 体能负迁移）？ | PCA + 聚类 | 探索性研究 |
| RQ4 | 同一球员在五人制 vs 十一人制中的行为剖面如何变化？ | 对比剖面分析 | 创新方法学 |

### 假设
- **H1**: 有五人制背景的球员在「第一脚触球质量」「小空间传球成功率」「决策速度」上显著优于纯十一人制球员
- **H2**: 五人制训练年限与迁移效应大小呈正相关，但存在收益递减（对数/平方根关系）
- **H3**: 转项年龄 < 14 岁的球员，迁移效应显著大于转项年龄 > 16 岁者（关键窗口期假说）
- **H4**: 迁移效应存在显著的个体间异质性（SD > 均值），可聚类为 3-5 种迁移模式

## 二、研究设计

### 设计类型
**回顾性横截面比较研究**（硕士阶段可行）→ 扩展为**前瞻性纵向追踪**（博士阶段）

### 样本量估算

```
效应量设定（基于 Oppici et al., 2019 和 Travassos et al., 2018）：
- 预期 Cohen's d = 0.50-0.80（中等偏大）
- α = 0.05, Power = 0.80
- 独立样本 t 检验：每组需要 51-128 人
- 考虑 20% 数据损耗：每组 61-154 人

实际招募目标：
├── 五人制背景组（FUT）：n = 80-120
│   ├── 入选标准：≥ 3 年五人制系统训练
│   ├── 亚组：业余级（n≥40）/ 省级+国家级（n≥40）
│   └── 亚组：早期转项 <14岁（n≥40）/ 晚期转项 ≥14岁（n≥40）
├── 纯十一人制对照组（CON）：n = 80-120
│   ├── 入选标准：无五人制系统训练（< 6 个月）
│   └── 与 FUT 组按年龄、十一人制年限、位置匹配
```

### 匹配策略（倾向性评分匹配 PSM）

```python
# 示意：用年龄、十一人制年限、位置匹配两组
from sklearn.linear_model import LogisticRegression
from sklearn.preprocessing import StandardScaler

# 第 1 步：预测"是否属于五人制组"的概率
ps_model = LogisticRegression()
ps_model.fit(X[['age', 'football_years', 'position_code', 'height', 'weight']], 
             y['is_futsal_group'])

# 第 2 步：1:1 最近邻匹配
ps_scores = ps_model.predict_proba(X)[:, 1]
matched_pairs = nearest_neighbor_matching(ps_scores, caliper=0.2)
```

## 三、变量操作化

### 因变量（迁移效果的指标）

| 维度 | 指标 | 数据来源 | 测量工具 |
|------|------|----------|----------|
| **技术执行** | 传球成功率（%） | 事件数据 | Wyscout / Opta / 视频标注 |
| | 第一脚触球评分（1-5） | 教练评分 | 李克特量表 |
| | 压力下传球成功率（%） | 事件数据 | 加标签「受压情境」 |
| | 射门转化率 | 事件数据 | 进球/射门 |
| **感知-认知** | 决策速度（秒） | 视频分析 | 接球到出球时间 |
| | 决策准确率（%） | 教练评审 | 3 名以上教练独立评分 |
| | 视觉扫描频率（次/分钟） | 视频编码 | 参考 Jordet 编码方案 |
| **体能表现** | 高强度跑距离 | GPS 数据 | Catapult / STATSports |
| | 加速度/减速度次数 | GPS 数据 | >3 m/s² |
| **战术理解** | PPDA（压迫强度） | 事件数据 | 对方传球数/本方防守动作数 |
| | 传球网络中心性 | 事件数据 | betweenness/closeness |

### 自变量（五人制经验特征）

| 变量 | 类型 | 编码 |
|------|------|------|
| 五人制训练年限 | 连续 | 年（精确到月再换算） |
| 五人制最高水平 | 序数 | 1=业余/2=省级/3=国家级/4=国际级 |
| 五人制位置 | 类别 | Pivot/Ala/Fixo/Goleiro |
| 转项年龄 | 连续 | 开始系统十一人制训练的年龄 |
| 双轨训练期长度 | 连续 | 同时训练两种赛制的月数 |
| 五人制训练密度 | 连续 | 小时/周 |
| 五人制比赛频率 | 连续 | 场次/月 |

### 控制变量

| 变量 | 编码 | 理由 |
|------|------|------|
| 年龄 | 连续 | 成熟度效应 |
| 十一人制训练年限 | 连续 | 排除「十一人制练得多」的替代解释 |
| 十一人制位置 | 类别 | 位置要求差异大 |
| 比赛级别 | 序数 | 联赛水平差异 |
| 对手实力（匹配场次） | 连续 | Elo / 联赛排名 |
| GPS 设备型号 | 类别 | 不同设备数据不可比 |

## 四、分析流水线（5 步）

### Step 1: 数据清洗 + 标准化

```python
# 伪代码框架
import pandas as pd
from sklearn.preprocessing import StandardScaler

# 1.1 缺失值处理
df = df.dropna(thresh=0.7)  # 删除缺失 >30% 的行
df = df.fillna(df.median())  # 中位数填充

# 1.2 异常值检测（IQR 方法）
Q1 = df.quantile(0.25)
Q3 = df.quantile(0.75)
IQR = Q3 - Q1
df = df[~((df < (Q1 - 1.5 * IQR)) | (df > (Q3 + 1.5 * IQR))).any(axis=1)]

# 1.3 比赛时长标准化（含补时按 90 分钟折算）
df['distance_per_90'] = df['total_distance'] / (df['match_minutes'] / 90)
```

### Step 2: 组间对比 + 效应量（RQ1）

```python
import pingouin as pg
from scipy import stats

# 2.1 匹配后组间比较
for metric in performance_metrics:
    d = pg.compute_effsize(df_fut[metric], df_con[metric], eftype='cohen')
    t, p = stats.ttest_ind(df_fut[metric], df_con[metric])
    print(f"{metric}: d={d:.3f}, p={p:.4f}")

# 2.2 多元线性混合模型（控制情境 + 匹配变量）
import statsmodels.formula.api as smf

model = smf.mixedlm(
    "performance_score ~ futsal_group * futsal_years + age + football_years + 
     position + match_level + (1|player_id)",
    data=df
)
result = model.fit()
```

### Step 3: 可解释 ML 预测迁移效应（RQ2）

```python
import xgboost as xgb
import shap

# 3.1 特征工程
X = df[[
    'futsal_years', 'futsal_level', 'futsal_position',
    'transition_age', 'concurrent_months', 'futsal_hours_per_week',
    'futsal_matches_per_month', 'age', 'football_years', 'position'
]]

# 3.2 XGBoost 回归
y = df['composite_performance_score']  # PCA 合成指标或单一指标

model = xgb.XGBRegressor(n_estimators=200, max_depth=5, learning_rate=0.05)
model.fit(X, y)

# 3.3 SHAP 解释
explainer = shap.TreeExplainer(model)
shap_values = explainer(X)
shap.summary_plot(shap_values, X, show=False)

# 3.4 输出：每个五人制经验特征的 SHAP 重要性排序
shap_importance = pd.DataFrame({
    'feature': X.columns,
    'shap_importance': np.abs(shap_values).mean(axis=0)
}).sort_values('shap_importance', ascending=False)
```

### Step 4: 迁移模式聚类（RQ3）

```python
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans

# 4.1 构建迁移差异矩阵
diff_matrix = compute_migration_difference(df)  # 每球员 vs 对照组均值的 z-score

# 4.2 PCA 降维
scaler = StandardScaler()
X_scaled = scaler.fit_transform(diff_matrix)
pca = PCA(n_components=0.75)
X_pca = pca.fit_transform(X_scaled)

# 4.3 K-Means 聚类 + 肘部法
from sklearn.metrics import silhouette_score
best_k = elbow_method(X_pca, k_range=range(2, 8))
kmeans = KMeans(n_clusters=best_k, random_state=42)
clusters = kmeans.fit_predict(X_pca)

# 4.4 解释聚类（每个簇的指标特征）
cluster_profiles = pd.DataFrame({
    f'Cluster_{i}': scaler.inverse_transform(
        pca.inverse_transform(kmeans.cluster_centers_[i])
    ) for i in range(best_k)
})
```

### Step 5: 跨赛制剖面对比（RQ4）

```python
# 5.1 双赛制行为剖面可视化
import matplotlib.pyplot as plt
import seaborn as sns

metrics_futsal = compute_player_profile(player_id, 'futsal')
metrics_football = compute_player_profile(player_id, 'football')

# 5.2 平行坐标图（同一球员两种赛制的指标归一化对比）
fig, ax = plt.subplots(figsize=(12, 6))
normalized_futsal = (metrics_futsal - mean_futsal) / std_futsal
normalized_football = (metrics_football - mean_football) / std_football
ax.plot(metric_names, normalized_futsal, 'r-', label='Futsal')
ax.plot(metric_names, normalized_football, 'b-', label='Football')

# 5.3 迁移保持指数（Bridging Behavior Index, BBI）
BBI = 1 - abs(normalized_futsal - normalized_football) / max(abs(normalized_futsal), 1)
```
