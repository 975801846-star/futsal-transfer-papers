## 🔧 方法学修正清单

| 原方案 | 问题 | 修正 |
|--------|------|------|
| 线性混合模型 (1|player_id) | 横截面不可辨识 | ANCOVA 或 (1|team_id) |
| 单独 PSM | 样本损失 + 不可观测混杂 | 双重稳健 IPTW 或 ANCOVA |
| XGBoost+SHAP 仅主模型 | n=200 不稳定，SHAP≠因果 | +DAG + E-value + 负对照，与 Ridge/LASSO 对比 |
| K-Means 仅轮廓系数 | 不可复现 | +Bootstrap ARI + Ward 层次聚类交叉验证 |
| RQ4 "迁移方向推断" | 剖面差异≠因果 | →"跨赛制剖面描述性对比" |
| Travassos 2018 效应量引用 | 该文是评论无 ES | → Oppici 2018 原始实证论文 |
