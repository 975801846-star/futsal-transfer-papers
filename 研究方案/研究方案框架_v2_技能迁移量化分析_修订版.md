## 二、研究设计

### 设计类型
**回顾性横截面比较**（硕士阶段）

### 样本量（修正）

| 场景 | 每组招募 | 总样本 |
|------|:------:|:------:|
| 核心 t 检验 d≥0.5，20% 损耗 | **80** | **160** |
| 含多重比较 m=5，20% 损耗 | **119** | **238** |

**推荐**：n=160-200 为核心分析，XGBoost 特征数 ≤15。

### 匹配策略（修正：双重稳健 IPTW 替代 PSM）

```python
# 第1步：倾向性评分加权（IPTW）
from sklearn.linear_model import LogisticRegression
ps_model = LogisticRegression()
ps_model.fit(X[['age', 'football_years', 'position_code']], y['is_futsal_group'])
ps_scores = ps_model.predict_proba(X)[:, 1]
weights = 1 / ps_scores  # 逆概率权重

# 第2步：加权 ANCOVA（双重稳健）
import statsmodels.api as sm
model = sm.WLS.from_formula(
    'performance_score ~ futsal_group + age + football_years + C(position)',
    weights=weights, data=df
).fit(cov_type='HC3')  # robust SE
```

**为什么不单独用 PSM**：n=200 是 PSM 下限，样本损失 + 不可观测混杂风险高。双重稳健 IPTW → 加权回归兼具两者优势。

## 三、变量操作化

### 因变量（多维因子得分）

| 维度 | 指标 | 测量 |
|------|------|------|
| **技术因子** | 传球成功率、压力下传球%、第一脚触球评分 | 事件数据 + 教练评分 |
| **决策因子** | 决策速度（ms）、决策准确率 | 视频编码（50/60fps） |
| **体能因子** | Accel/min、Decel/min、HSR 距离% | GPS（同品牌设备） |
| **战术因子** | PPDA、传球网络中心性 | 事件数据衍生 |

**复合评分构建（修正）**：CFA 验证性因子分析 → 2-4 维度因子得分 → multi-output XGBoost。**不再用 PCA 合成单一综合分**（信息丢失 + 可解释性崩塌）。

### 自变量（五人制经验）

| 变量 | 类型 | 编码 |
|------|------|------|
| 五人制训练年限 | 连续 | 年（精确到月） |
| 五人制最高水平 | 序数 | 业余/省级/国家级/国际级 |
| 五人制位置 | 类别 | Pivot/Ala/Fixo（排除门将） |
| 转项年龄 | 连续 | 开始系统十一人制训练年龄 |
| 双轨训练期长度 | 连续 | 同时训练两种赛制月数 |
| 五人制训练密度 | 连续 | 小时/周 |

### 控制变量 + 竞争假设控制

| 变量 | 理由 |
|------|------|
| 年龄 | 成熟度效应 |
| 十一人制训练年限 | 排除"练得多所以好" |
| 十一人制位置 | 位置要求差异 |
| **相对年龄**（出生季度） | 选择偏差 |
| **总训练时数** | 训练量≠训练种类效应 |
| **SSG 训练比例** | 最强竞争假设 |

## 五、新增：GPS 最小指标集（MIS-6）

| 优先级 | 指标 | 单位 | 反映的迁移维度 |
|:------:|------|------|---------------|
| P0 | 加速度次数 Accel >2.5 m/s² | 次/min | 爆发性启动能力 |
| P0 | 减速度次数 Decel < -2.5 m/s² | 次/min | 制动/变向能力 |
| P0 | 高强度跑距离占比 >19.8 km/h | %总距离 | 高强度跑效率 |
| P1 | 代谢功率 | W/kg | 综合运动负荷 |
| P1 | PlayerLoad™ | au/min | 全身累积负荷 |
| P2 | 总跑动距离 | m per 90 | 基础体能协变量 |

**逻辑**：迁移检测关注**单位时间高强度动作频率**，非总跑量。同品牌设备分析加速度指标。

## 六、新增：事件编码协议

- **设计**：自动数据（范围） + 手动深度编码（精度）
- **决策速度**：T_release − T_control，50/60fps 视频
- **编码员信度**：Cohen's κ ≥ 0.80, ICC ≥ 0.75
- **工作量**：C 类编码 4-6 小时/场，双重编码 10-20% 样本

## 七、新增：教练评分协议

- **评分者**：5 名教练，7 点李克特 + BARS 行为锚定
- **培训**：2-3 小时校准 + 5-10 锚定视频
- **信度**：ICC(2,k) ≥ 0.80, G-study Φ ≥ 0.80
- **模式**：视频锚定评分（非现场）

## 八、效应量汇报标准（修正）

| 汇报内容 | 规范 |
|----------|------|
